# Multi-Omics Data Imputation Pipeline (miRNA → DNA)

This repository contains an end-to-end deep learning framework designed to impute missing DNA features using microRNA (miRNA) expression profiles across multi-cancer datasets (**BRCA**, **BLCA**, and **OV**).

The pipeline automatically handles cross-validation splits, feature scaling, model training, performance evaluation, missing data imputation, and confidence score estimation.
